RawCredentials 🔒
Field names match the live Sentinel API response (Python reference: sentinel_api.py register_run docstring). expiration is the documented Python name; expires_at is accepted as an alias in case the live API uses a different casing. The field is optional so a missing value does not abort the run – it falls back to a far-future timestamp (credentials treated as always-fresh).
